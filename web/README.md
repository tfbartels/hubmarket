# Hub Market - Web

## Iniciando


### Pré-requisitos

```
NodeJs versão >= 8.9.0
NPM versão >= 5.6.0
Git
Quasar
VueJs
```

### Instalação

Entre na pasta web do projeto e instale as dependências com: 
```
npm install
```

### Execução

Rode a aplicação com o comando:
```
npm run-script dev
```
A aplicação estará disponivel na URL http://localhost:8080

## Deploy

```
npm run-script build
```
